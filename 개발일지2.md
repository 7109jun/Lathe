# Lathe Development Log #2

## 진행 상황

오늘은 Lathe의 전체 구조를 설계했습니다.

### 프로젝트 구조

* `.mn` (Manager)
* `.co` (Creator)
* `.lt` (Lathe)

각 언어는 서로 다른 역할을 담당하도록 설계하여 유지보수성과 확장성을 높이는 것을 목표로 합니다.

---

## MN (Manager)

프로젝트와 데이터를 관리하는 핵심 시스템입니다.

기능

* 프로젝트 관리
* 오브젝트 데이터 관리
* 저장 데이터 관리
* FPS 설정
* 엔진 설정
* 메모리 관리
* 최적화

---

## CO (Creator)

사용자가 보는 화면을 구성합니다.

기능

* UI
* 레이아웃
* 색상
* 오브젝트 위치
* 크기
* 화면 요소 제어

---

## LT (Lathe)

게임의 내부 동작을 담당하는 스크립트 언어입니다.

기능

* 이벤트
* 함수
* AI
* 조건문
* 반복문
* 게임 로직

---

## IDE 설계

오늘 확정된 기능

* 새 프로젝트
* 저장
* RUN
* STOP
* 오브젝트 생성
* 오브젝트 목록
* Scene View
